# S-Blank (Віртуальний бланк відповідей)

**S-Blank** — це зручне браузерне розширення та скрипт для комфортного заповнення бланків відповідей під час проходження тестів.

## Основні можливості

- **Кастомізація:** Підтримка декількох тем оформлення (Світла, Темна, Тепла, Космічна) та налаштування розміру шрифту.
- **Режим викладача:** Дозволяє вводити правильні відповіді, зберігати їх як ключі та автоматично перевіряти результати учнів.
- **Таймер:** Вбудований таймер для контролю часу під час виконання тестів.
- **Історія результатів:** Автоматичне збереження результатів (кількість правильних/неправильних відповідей, витрачений час) для подальшого аналізу.
- **Звукові сигнали:** Сповіщення під час перевірки відповідей (правильно/неправильно).
- **Зручний інтерфейс:** Компактна панель, яку можна згортати, розгортати та переміщувати по екрану.

## Про розробку

Цей проєкт є результатом **спільної роботи людини та штучного інтелекту (ШІ)**. 
Я (автор) генерував ідеї, визначав логіку роботи, структуру та дизайн, а ШІ виступав у ролі помічника для написання, рефакторингу та оптимізації коду. Це чудовий приклад того, як людина та нейромережі можуть співпрацювати для створення корисних інструментів!

## Встановлення

1. Завантажте архів з розширенням або склонуйте цей репозиторій.
2. Відкрийте браузер (Chrome, Edge тощо) та перейдіть на сторінку розширень (`chrome://extensions/`).
3. Увімкніть **Режим розробника** (Developer mode) у правому верхньому куті.
4. Натисніть **Завантажити розпаковане розширення** (Load unpacked) та оберіть папку `extension` з цього репозиторію.
5. Готово! Тепер ви можете використовувати розширення.

*(Також у репозиторії доступний UserScript версія `Віртуальний_blank_відповідей_PRO_+_Самоперевірка_4_6.js`, яку можна встановити через Tampermonkey)*
